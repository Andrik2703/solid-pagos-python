
# 🐍 Sistema de Pagos con Principios SOLID

## Descripción
Proyecto educativo que demuestra la aplicación de los 5 principios SOLID
en un sistema de procesamiento de pagos en Python.

## Principios Aplicados
- **SRP**: Clases con responsabilidad única
- **OCP**: Extensible sin modificar código existente
- **LSP**: Métodos de pago intercambiables
- **ISP**: Interfaz pequeña y específica
- **DIP**: Dependencia de abstracciones

## Ejecución
```bash
python pagos_solid.py
